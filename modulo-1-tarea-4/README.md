## Descripción del proyecto
Proyecto desarrollado con **React** utilizando **Vite** como entorno de desarrollo.
La aplicación consiste en la creación de un componente funcional reutilizable llamado **Tarjeta**, el cual recibe información mediante **props** y se renderiza múltiples veces desde el componente principal **App** para mostrar distintas profesiones.

El objetivo del proyecto es aplicar conceptos fundamentales de React como:
- Componentes funcionales
- Uso de props
- JSX
- Estilos con CSS
- Organización de un proyecto profesional

## Tecnologías utilizadas
- React
- Vite
- JavaScript
- CSS
